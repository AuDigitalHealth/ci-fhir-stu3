SD.6 StructureDefinition-practitioner-ident-1

Practitioner with Mandatory Identifier [Draft]

Implementation guidance

No additional guidance is suggested to support implementation of this profile for the expected usage scenarios of this implementation guide.

SD.6.1 Formal Views of Profile Content

The official URL for this profile is: 

http://ns.electronichealth.net.au/ci/fhir/3.0/StructureDefinition/practitioner-ident-1

The purpose of this profile is to define a representation of a practitioner in an Australian context where an identifier is mandatory.

Must Support

In the context of this profile Must Support SHALL be interpreted as follows:

  • The system SHALL be able to include the element when authoring a resource where data is available

  • The system SHALL be able to store and retrieve the element

  • The system SHALL be able to process resources containing the element

This profile builds on AUBasePractitioner.

This profile was published on Fri May 31 11:37:30 AEST 2019 as a draft by Australian Digital Health Agency.
